# Teknofest Hava Savunma Sistemi (Air Defense System)

Bu proje, 2024 yılında Teknofest yarışması kapsamında geliştirilmiştir. Projenin temel amacı, görüntü işleme tekniklerini kullanarak hava araçlarının tespiti ve takibi üzerine bir çözüm üretmektir.

## 🚀 Proje Hakkında
Bu çalışma kapsamında aşağıdaki teknikler ve araçlar kullanılmıştır:
* **Görüntü İşleme:** OpenCV kütüphanesi ile nesne tespiti.
* **Algoritmalar:** Piksel operasyonları, kontur tespiti ve Hough dönüşümü.
* **Dil:** Python.

## 🛠️ Özellikler
* Gerçek zamanlı nesne tespiti.
* Belirli bir mesafedeki hedeflerin koordinat takibi.
* Gürültü engelleme filtreleri ile iyileştirilmiş görüntü analizi.
* Spesifik Tespit: Renk eşikleme ve dairesellik analizi kullanarak kırmızı yuvarlak balonları tespit eden ve takip eden algoritma başarıyla uygulanmıştır.

## 📝 Notlar
Bu proje 2024 yılında Teknofest sürecine hazırlık aşamasında geliştirilmiş olup, yarışmanın o seneki aşamalarında finale kalmadan elenmiştir. Ancak akademik ve teknik gelişim açısından önemli bir temel teşkil etmektedir.

## English

### 🚀 Project Overview
This project was developed for the **2024 Teknofest** competition. It focuses on object detection and tracking for aerial vehicles using advanced computer vision techniques.

### 🛠 Technical Stack
* **Language:** Python
* **Libraries:** OpenCV, NumPy
* **Core Concepts:** Pixel operations, contour detection, Hough Transform, and real-time image processing.

### 📁 Key Features
* Real-time detection of aerial targets.
* Coordinate tracking using contour analysis.
* Enhanced image analysis with noise reduction filters.
* Specific Detection: Successfully implemented an algorithm to detect and track red circular balloons using color thresholding and circularity analysis.

### 📌 Note
This project was completed in 2024. Although it reached the elimination stages of Teknofest, it did not qualify for the finals. It remains a core part of my technical development in computer vision.
